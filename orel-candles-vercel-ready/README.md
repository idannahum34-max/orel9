# OREL Candles Website

אתר React + Vite מוכן להעלאה ל-GitHub ולפריסה ב-Vercel.

## הרצה מקומית

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## העלאה ל-GitHub

```bash
git init
git add .
git commit -m "Initial OREL website"
git branch -M main
git remote add origin YOUR_GITHUB_REPO_URL
git push -u origin main
```

## פריסה ב-Vercel

1. נכנסים ל-Vercel.
2. לוחצים Add New Project.
3. בוחרים את ה-repository מ-GitHub.
4. מוודאים:
   - Framework Preset: Vite
   - Build Command: npm run build
   - Output Directory: dist
5. לוחצים Deploy.
