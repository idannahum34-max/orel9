# OREL Candles — Vercel Ready

גרסה נקייה ומהירה לפריסה ב-Vercel.

## Vercel settings

- Framework Preset: Vite
- Install Command: npm install
- Build Command: npm run build
- Output Directory: dist

## Local run

```bash
npm install
npm run dev
```

## Upload to GitHub

Upload the folder `orel-candles-vercel-fast` to GitHub, then import the repository in Vercel.
